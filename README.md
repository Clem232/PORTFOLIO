# Front – Jeu interactif Three.js

## Prérequis

- Node.js 18+

## Installation

```bash
npm install
```

## Lancer en développement

```bash
npm run dev
# → http://localhost:5173
```

## Build de production

```bash
npm run build
npm run preview
```

## Structure

```
front/
├── index.html   # Point d'entrée HTML
├── main.js      # Scène Three.js principale
└── package.json
```
